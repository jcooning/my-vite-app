## 1. Windows PC에서 할 일

### ✅ 체크리스트
- [ ] 프로젝트 폴더 압축
- [ ] .env 파일 내용 확인 및 백업
- [ ] 압축 파일 맥북으로 전송

### 1-1. 프로젝트 폴더 압축하기

**단계별 설명:**

```
1️⃣ 파일 탐색기 열기
   - 키보드: Windows 키 + E

2️⃣ 프로젝트 폴더로 이동
   - 주소: D:\Study\my-vite-app

3️⃣ 폴더 압축하기
   - my-vite-app 폴더에서 마우스 오른쪽 클릭
   - "압축" 또는 "Send to" → "Compressed (zipped) folder" 선택
   - my-vite-app.zip 파일이 생성됩니다

4️⃣ 압축 파일 크기 확인
   - 약 50-100MB 정도면 정상입니다
   - 만약 500MB 이상이면 node_modules 폴더가 포함된 것입니다
```

### 1-2. 환경 변수 파일 백업

**중요! 이 파일이 없으면 프로그램이 작동하지 않습니다!**

```
1️⃣ .env 파일 찾기
   - D:\Study\my-vite-app\.env

2️⃣ 메모장으로 열기
   - .env 파일을 마우스 오른쪽 클릭
   - "연결 프로그램" → "메모장" 선택

3️⃣ 내용 복사하기
   - Ctrl + A (전체 선택)
   - Ctrl + C (복사)
   - 카카오톡 나에게 보내기나 이메일로 전송
```

**예시 내용:**
```
VITE_KAKAO_API_KEY=당신의_카카오_API_키
VITE_NOTION_API_KEY=당신의_노션_API_키
VITE_NOTION_DATABASE_ID=당신의_노션_데이터베이스_ID
```

## 2. 파일 옮기기

### 방법 1: AirDrop (가장 쉽고 빠름!) ⭐ 추천

**Windows PC:**
```
❌ Windows에서는 AirDrop을 직접 사용할 수 없습니다
→ 방법 2, 3, 4 중 선택하세요
```

### 방법 2: USB 메모리

```
1️⃣ USB 메모리 꽂기
2️⃣ my-vite-app.zip 파일을 USB로 복사
3️⃣ USB를 맥북에 꽂기
4️⃣ 파일을 맥북으로 복사
```

### 방법 3: 클라우드 (Google Drive, iCloud)

**Google Drive 사용:**
```
1️⃣ drive.google.com 접속
2️⃣ 로그인
3️⃣ "새로 만들기" → "파일 업로드"
4️⃣ my-vite-app.zip 선택
5️⃣ 업로드 완료 대기 (5-10분)
6️⃣ 맥북에서 같은 계정으로 접속하여 다운로드
```

### 방법 4: 이메일

```
1️⃣ 이메일 작성
2️⃣ 받는 사람: 자신의 이메일 주소
3️⃣ 첨부파일: my-vite-app.zip
4️⃣ 전송
5️⃣ 맥북에서 이메일 확인 후 다운로드
```

## 4. 프로그램 설치

### 4-1. Homebrew 설치 (필수!)

**Homebrew란?** 맥에서 프로그램을 쉽게 설치할 수 있게 해주는 도구

**설치 방법:**

```bash
1️⃣ 터미널 열기 (⌘ + Space → "터미널")

2️⃣ 아래 명령어를 복사해서 붙여넣기:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

3️⃣ Enter 키 누르기

4️⃣ 비밀번호 입력
   ⚠️ 입력해도 화면에 안 보이지만 정상입니다!
   그냥 비밀번호 입력하고 Enter

5️⃣ 설치 완료까지 기다리기 (3-5분)
   "Installation successful!" 메시지가 나오면 성공!
```

**설치 확인:**
```bash
brew --version
```
버전 번호가 나오면 성공! ✅

### 4-2. Node.js 설치 (필수!)

**Node.js란?** JavaScript를 실행할 수 있게 해주는 프로그램

**설치 방법:**

```bash
1️⃣ 터미널에서 입력:

brew install node

2️⃣ 설치 완료 대기 (3-5분)

3️⃣ 설치 확인:

node --version
npm --version

4️⃣ 버전 번호가 나오면 성공!
   예: v20.11.0
```

### 4-3. Visual Studio Code 설치 (선택사항)

**VS Code란?** 코드를 편집할 수 있는 프로그램 (메모장의 고급 버전)

**방법 1: Homebrew로 설치 (추천)**
```bash
brew install --cask visual-studio-code
```

**방법 2: 직접 다운로드**
```
1️⃣ https://code.visualstudio.com 접속
2️⃣ "Download Mac Universal" 클릭
3️⃣ 다운로드된 파일을 더블클릭
4️⃣ Visual Studio Code.app을 "응용 프로그램" 폴더로 드래그
```

### 5-1. 압축 파일 풀기

```

   - 보통 "다운로드" 폴더에 있습니다
```

## 6. 문제 해결

### ❌ "npm을 찾을 수 없습니다"

**원인:** Node.js가 제대로 설치되지 않음

**해결:**
```bash
# Node.js 재설치
brew uninstall node
brew install node

# 확인
node --version
npm --version
```

### ❌ "권한이 거부되었습니다" (Permission denied)

**해결:**
```bash
sudo npm install
```
비밀번호 입력 (화면에 안 보여도 정상)

### ❌ "포트가 이미 사용 중입니다"

**원인:** 이미 실행 중인 프로그램이 있음

**해결:**
```bash
# 실행 중인 프로세스 종료
lsof -ti:3001 | xargs kill
lsof -ti:5173 | xargs kill

# 다시 실행
node notion-proxy.js
npm run dev
```

### ❌ "command not found: brew"

**원인:** Homebrew가 설치되지 않음

**해결:** 4-1단계로 돌아가서 Homebrew 재설치

### ❌ ".env 파일을 찾을 수 없습니다"

**해결:**
```bash
# 현재 위치 확인
pwd

# .env 파일 생성
nano .env

# 내용 입력 후 저장 (Ctrl+O, Enter, Ctrl+X)
```

### ❌ "Module not found" 에러

**원인:** 패키지가 제대로 설치되지 않음

**해결:**
```bash
# node_modules 폴더 삭제
rm -rf node_modules

# 다시 설치
npm install
```
